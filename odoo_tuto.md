To create module

custom modules/
└── hotel_management/
    ├── __manifest.py__
    └── __init.py__

add json format in manifest
{
    'name':'Hotel Management'
}

turn on active developer mode

update app list

custom modules/
├── hotel_management
└── static/
    ├── description/
    │   └── icon.png
    ├── __manifest.py__
    └── __init.py__

then upgrade the module
